# Versão 1: Modelo Inicial Estático

Esta é a base histórica do SalesTrack. O foco aqui foi a definição da interface e das regras de negócio primárias sem a complexidade de frameworks.

### 🛠 Aspectos Técnicos
- **Arquitetura:** Camada única (Modelo simples).
- **Frontend:** Desenvolvido com HTML5, CSS3 e JavaScript "bruto". Não há componentização; cada página carrega seus próprios recursos de forma rígida.
- **Backend/Dados:** Nesta fase, a lógica de persistência era simplificada, com os scripts de banco de dados SQL e arquivos de lógica centralizados em uma única estrutura, sem separação de pastas por responsabilidades.

**Objetivo alcançado:** Validação do fluxo de usuário e design do sistema.
